Lexique :

- Espace disque : c'est le nombre de Mo que votre compte peut contenir. Pour vérifier si l'espace disque proposé est suffisant, regardez la taille du répertoire de votre site en local.

- Bande passante : c'est le nombre de Ko consommé via http et ftp. Par exemple, si vous uploadez votre site (13 Mo par exemple), vous consommez 0.013 Go de bande passante. Si 100 visiteurs visitent 10 pages votre site web, et que vos pages font en moyenne 10 Ko, vous consommez alors 10 (pages) x 10 (kilo) x 100 (visiteurs) = 1000 Mo soit 1 Go.

- Fonction Php4 : vous avez un site dynamique programmé en php ? L'option Php est activée dès que votre compte est crée !

- Base de donnée MySql : la plupart du temps, les sites dynamiques sont combinés entre Php et MySql. La base de donnée qui vous est attribuée est à tout moment accessible par PhpMyAdmin.

- Répertoire Cgi-Bin : si vous utilisez des scripts Perl, vous devez mettre vos fichiers .cgi et .pl dans ce répertoire.

- Streaming Real Audio/Vidéo : vous souhaitez diffuser des extraits sonore ou vidéo ? Vous pouvez le faire en utilisant des fichiers au format .rm ou .ram

- Accès Ftp : pour mettre en ligne votre site, vous devrez passer par un logiciel ftp. La mise en ligne de votre site web ne se fera pas par gestionnaire de fichiers.

- Hébergement en sous-domaine : votre site n'aura pas d'adresse du type : http://www.site_a_adresse_longue.com/votre_login/ mais http://votre_login.host-php.org pour avoir une adresse facile à mémoriser.

- Panneau d'administration Cpanel : c'est vous qui allez tout gérer grâce à cette interface complète. Vous pouvez gérer vos emails, base de données mysql, mot de passe du compte , (...)
